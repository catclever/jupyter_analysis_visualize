⚠️  常见问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q1：Permission Denied / [WinError 5] 拒绝访问
├─ 原因：Windows 权限问题
└─ 解决：以管理员身份运行命令提示符或 Git Bash

Q2：python not found
├─ 原因：Python 3.10 未安装或未添加到 PATH
└─ 解决：
   1. 下载并安装 Python 3.10
   2. 安装时勾选 "Add Python to PATH"
   3. 重启电脑后再试

Q3：无法访问 http://localhost:8000/
├─ 原因：端口被占用或服务未成功启动
└─ 解决：
   1. 查看命令提示符窗口是否有错误信息
   2. 试试改端口：编辑 backend/app.py 最后几行，改 port=8001
   3. 重新运行脚本

Q4：首次启动很慢
├─ 原因：正常！虚拟环境创建和依赖安装需要时间
└─ 预期：约 5-10 分钟，取决于磁盘速度

Q5：我在 Git Bash 中，怎么用？
└─ 查看：GIT_BASH_GUIDE.md
